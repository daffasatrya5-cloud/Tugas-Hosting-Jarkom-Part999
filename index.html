Mata Kuliah: Jaringan Komputer

Dalam sebuah infrastruktur jaringan, setiap node (komputer) memerlukan Network Interface Card (NIC) sebagai antarmuka fisik untuk dapat mentransmisikan sinyal data. Di lingkup lokal (LAN), distribusi data dikelola oleh Switch, yang bekerja pada Data Link Layer untuk meneruskan paket data secara presisi ke alamat fisik (MAC Address) tujuan, sehingga mencegah tabrakan data (collision).
